## Tworzenie środowiska wirtualnego venv:

## Windows:
- Utwórz środowisko wirtualne za pomocą polecenia: 
> python -m venv venv
- Aktywuj środowisko wirtualne za pomocą polecenia: 
> venv\Scripts\activate

## MacOS:
- Utwórz środowisko wirtualne za pomocą polecenia:
> python3 -m venv venv
- Aktywuj środowisko wirtualne za pomocą polecenia:
> source venv/bin/activate
